## Instruções para Implementação

Para implementar este design, são necessárias as seguintes informações do Figma:

### 1. Especificações de Design Necessárias
- [ ] Screenshots ou exports das telas principais
- [ ] Guia de cores (palette completo)
- [ ] Guia de tipografia (fontes, tamanhos, weights)
- [ ] Espaçamentos e grid system
- [ ] Componentes UI identificados
- [ ] Interações e animações especificadas
- [ ] Breakpoints responsivos

### 2. Como Obter as Especificações

#### Opção A: Export Manual do Figma
1. Abra o arquivo no Figma
2. Selecione os frames/componentes desejados
3. Exporte como PNG/SVG para referência visual
4. Copie especificações CSS do Dev Mode
5. Salve os arquivos nesta pasta (`figma/`)

#### Opção B: Compartilhar Acesso
1. No Figma, clique em "Share"
2. Copie o link público (se disponível)
3. Ou forneça acesso de visualização

#### Opção C: Figma Dev Mode
1. Ative o Dev Mode no Figma (ícone </> no canto superior direito)
2. Selecione os elementos e copie as especificações CSS
3. Exporte assets e ícones necessários
4. Documente componentes reutilizáveis
