# Retrobox Argentina - Quick Start

## Instalação

```bash
cd retrobox-argentina
npm install
npm run dev
```

Acesse: http://localhost:3000

## Estrutura

- `src/app/` - Páginas
- `src/components/` - Componentes React
- `src/context/` - Contextos (Cart, Wishlist)
- `public/images/` - Imagens estáticas

## Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- Embla Carousel

## Notas

- JavaScript (JSX) apenas - SEM TypeScript
- Design monocromático (preto, branco, gradientes)
- Mobile-first responsivo
